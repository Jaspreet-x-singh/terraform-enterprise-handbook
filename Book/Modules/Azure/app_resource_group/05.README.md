# Resource Group Module

This Terraform module creates an Azure Resource Group with optional tagging support.  
It is designed to be lightweight, reusable, and consistent with production‑grade module standards.

---

## 📦 Module Structure

  modules/resource_group/
  ├── main.tf
  ├── variables.tf
  ├── outputs.tf
  └── README.md

---

## Inputs

| Name                       | Type                                                                 | Default | Description |
|----------------------------|----------------------------------------------------------------------|---------|-------------|
| name                       | string                                                               | n/a     | Name of the App Service Plan |
| resource_group_name        | string                                                               | n/a     | Resource group name |
| location                   | string                                                               | n/a     | Azure region |
| tags                       | map(string)                                                          | {}      | Resource tags |

---

## 🚀 Usage Example

```hcl
module "resource_group" {
  source   = "../../modules/resource_group"

  name     = var.resource_group_name
  location = var.location
  tags     = var.tags
}
