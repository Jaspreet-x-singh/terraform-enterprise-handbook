# Linux App Service Module

This module deploys an **Azure Linux Web App** using an existing **App Service Plan**.

## Features

- Linux Web App (`azurerm_linux_web_app`)
- Supports:
  - Managed Identity
  - App Settings
  - Connection Strings
  - Logging
  - HTTPS-only
  - Client Affinity
  - Site Config (TLS, Always On, HTTP2, Health Check)
  - Timeouts
  - Tags

---

## Usage

```hcl
module "linux_app_service" {
  source = "github.com/your-org/terraform-azure-modules//linux_app_service"

  name                = "my-linux-app"
  resource_group_name = module.app_resource_group.name
  location            = module.app_resource_group.location
  app_service_plan_id = module.app_service_plan.id

  app_settings = {
    "WEBSITE_RUN_FROM_PACKAGE" = "1"
  }

  site_config = {
    always_on           = true
    ftps_state          = "Disabled"
    minimum_tls_version = "1.2"
    http2_enabled       = true
    health_check_path   = "/health"
    python_version      = "3.10"
    node_version        = null
    dotnet_version      = null
    java_version        = null
  }

  tags = {
    environment = "dev"
    owner       = "platform-team"
  }
}
