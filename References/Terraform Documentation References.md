# Terraform Documentation References  
Useful links for following along with official Terraform resources.

## 🔗 Official Terraform Registry (HashiCorp Namespace)

**URL:**  
https://registry.terraform.io/namespaces/hashicorp

This is the authoritative source for:

- Official Terraform **providers**
- Official Terraform **modules**
- Provider documentation (arguments, attributes, examples)
- Resource and data source references
- Version history and changelogs
- Authentication and configuration guides

## ⭐ Why This Reference Matters

The Terraform Registry is your **single source of truth** for:

- Correct argument names  
- Supported SKUs  
- Required fields  
- Deprecations  
- Provider-specific quirks  

It ensures your modules stay aligned with official provider behavior, especially for:

- Private endpoints  
- Managed identities  
- Diagnostic settings  
- Customer-managed keys  
- Autoscaling  
- Network rules
